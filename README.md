# Refined Recipe

![plot](./images/refined-recipe.png)

### Project Motivation
- Wanted to build app around Google Firebase
- Functioning login/sign up features
- Different users can store different data
- Build an app to store my personal recipes

### Features
- User can sign up or log in to an account
- User can update their profile information
- User can view new recipe for them to learn
- User can add a recipe to their personal recipe book
- Can edit or delete personal recipe
